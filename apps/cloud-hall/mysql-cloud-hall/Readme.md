## 运行镜像，进入容器

```shell
docker run --name -p 8096:8096 cloud-hall -d 3a1ca885e154
docker exec -it 101a5abc0fe2 /bin/bash
```


## 容器中启动redis-server

```shell
apt-get update -y
apt-get install redis-server
```

```shell
redis-server
```
